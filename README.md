# Objetos con Intención

Este taller está compuesto por una serie de ejercicios que tienen como fin ser una introducción al aprendizaje de programación de componentes electrónicos, con el fin de brindar un hilo conector que permita a diseñadores u otras disciplinas afines empaparse con las posibilidades de creación que abre tener estas herramientas a nuestra disposición.

## 1- El Espejo del Sistema
Antes de que un objeto actúe, debe aprender a comunicarse. Este módulo trata sobre la creación de un puente de entendimiento entre la lógica interna del microcontrolador y el lenguaje humano. No programamos luces, programamos flujos de interacción que podemos monitorear y diagnosticar en tiempo real.

**Concepto:** Visibilizar el pensamiento digital.

**Objetivo:** Comprender la estructura básica de un programa (setup y loop) y utilizar el monitor serial para aprender a comunicarnos con nuestro hardware.

## 2- Sentir el Entorno
Los sensores no son solo piezas electrónicas; son traductores de fenómenos físicos (luz, distancia, tacto) a datos numéricos. Aprenderemos a capturar la sutileza del mundo de los sensores y convertirla en una escala que el diseño pueda comprender, visualizar y manipular a su antojo.

**Concepto**: El hardware como órganos sensoriales.

**Objetivo:** Dominar la lectura de entradas (inputs) analógicas y digitales, y aplicar el concepto de mapeo para transformar rangos de datos crudos en valores útiles para una interfaz.

## 3- Acción y Reacción
Un diseño inteligente es aquel que reacciona. Aquí cerramos el bucle: el dato recolectado se convierte en una acción física. Exploramos cómo el código puede controlar la energía para mover un brazo mecánico o variar la intensidad de una luz, transformando la información en una experiencia tangible.

**Concepto:** El hardware como un elemento vivo, capaz de reaccionar ante la información que recibe.

**Objetivo:** Combinar el sensor del ejercicio anterior con un actuador; crear una respuesta física ante un estímulo ambiental.

## 4- Objetos sin ataduras
El diseño ya no termina en el objeto físico. En este módulo, el objeto trasciende su ubicación y se conecta al vasto multiverso de Internet. Aprenderemos que un sensor en una habitación puede activar un evento al otro lado del mundo, convirtiendo un producto aislado en un servicio conectado.

**Concepto:** El objeto como parte de una red.

**Objetivo:** Configurar una conexión Wi-Fi básica para comunicar datos desde la nube con el mundo digital o con otros objetos, introduciendo los conceptos de protocolos de red y dashboards de control remoto.
