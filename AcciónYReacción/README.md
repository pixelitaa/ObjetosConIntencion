# Acción y Reacción
Un diseño inteligente es aquel que reacciona. Aquí cerramos el bucle: el dato recolectado se convierte en una acción física. Exploramos cómo el código puede controlar la energía para mover un brazo mecánico o variar la intensidad de una luz, transformando la información en una experiencia tangible.

## El Puente entre Sentir y Actuar
En los módulos anteriores aprendimos a "leer" el mundo. Ahora vamos a "escribir" en él. Para un diseñador, un actuador es el componente que permite que el objeto se comunique sin usar palabras ni pantallas, sino mediante movimiento, luz o sonido.

**El Servomotor: Movimiento Controlado** A diferencia de un motor común que gira sin parar, un Servomotor tipo de motor que funciona con precisión decirle exactamente en qué ángulo posicionarse (de 0° a 180°).

**El LED y el PWM: Intensidad Variable** No solo podemos encender o apagar una luz. Mediante una técnica llamada PWM (Pulse Width Modulation), el microcontrolador puede simular estados intermedios, la tan aclamada "Luz tenue"
